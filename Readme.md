ansible-dockerhost
===============

An ansible role that installs docker host on ubuntu

### Role Variables
```yaml
dockerhost_group                : "docker"
docker_compose_version          : "1.25.3"
docker_compose_dir              : "/opt/docker_compose"
docker_compose_install          : True
docker_daemon_config            : {}
```
